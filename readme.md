# Magic Curtain   
遺伝的アルゴリズムを用いた   
個性的なカーテン柄自動生成アプリ   

## 開発環境
 * Python 3.6.0 (numpy 1.11.3)
 * pygame 1.9.3 [[参考動画]](https://www.youtube.com/watch?v=ki_5uS4bOgQ) [[download link]](http://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame)
